# 星佑教育 課程管理系統 — 功能介紹及使用手冊

> 線上網址（示範）：https://course-cms-nine.vercel.app
> UI 語言：繁體中文。後台（管理員）為電腦版；導師打卡為手機版。

---

## 一、系統概覽

系統分為兩部分：

| 部分 | 對象 | 裝置 | 登入方式 | 網址 |
|---|---|---|---|---|
| 管理後台 | 管理員 / 職員 | 電腦 | 電郵 + 密碼 | `/login` |
| 導師打卡系統 | 導師 | 手機 | 電話 + SMS 驗證碼 | `/tutor-login` |

示範帳戶：
- 管理員：`admin@cms.local` / `password123`
- 導師：電話 `91234567`（陳大文）等 6 位，密碼 `password123`

## 二、功能介紹（後台）

### 1. 學校與課程（主界面）
- 以**學年**分頁瀏覽（可新增 / 隱藏學年）。
- **日期範圍**篩選：標示該範圍內有課堂的課程。
- 以**學校 → 課程卡**顯示；可搜尋學校名 / 課程名。
- 每張課程卡顯示**狀態圓點**（見第五節）。
- 可**新增學校**（含打卡地理範圍）、**新增課程**。

### 2. 課程頁（四個分頁）
- **課程基本信息**：學校資料、課程資料、導師要求、課程要求、**課程物料**、**小組**。可編輯課程、複製課程、刪除課程。
- **課堂詳情**：逐堂列表，可**即時編輯**（上課時間、課程準備、導師備課、導師、導師打卡時間、導師費、備註），可停用 / 啟用課堂、批量產生課堂。
- **批量更新**：選操作（更換導師 / 更換代課導師 / 導師備課 / 更換費用）→ 選小組 → 選日期 → 設定數值 → 保存；更換導師時可**製作 JC（工作確認書）**。
- **工作確認書**：列出此課程已製作的確認書。

### 3. 小組（一課程可多組）
每組可設：名稱、學生年級（多選）、人數、上堂地點、物資擺放位置、**上課星期（可多選）**、課堂時間、預算、要求助教 / 助理。

### 4. 課程物料
逐項物料（名稱、數量），狀態一鍵切換：未準備 / 準備中 / 已準備。

### 5. 上堂提醒
選一個**日期**，顯示當天所有上課的課程，按學校 → 課程分組，列出小組、時間、導師、電話（代課另列）。

### 6. 課程進度表
整個學年的**物資狀態**與**導師備課**進度，以彩色格顯示。

### 7. 工作計劃表
顯示某月份仍有**物資未準備好**的課程，列出最早未完成日期、課堂代號。可按**更新狀態**批量設定物資準備進度。

### 8. 導師請假表
某時段內**空缺 / 有代課 / 更換導師**的課堂安排一覽。

### 9. 導師搜索
按導師顯示其於該學年所任教的所有課堂（含代課），可搜尋、選學年。

### 10. 糧單
選月份，按導師結算**堂費**（依打卡的薪金規則），顯示每位小計及總計，可**匯出 CSV / XLSX**（含銀行戶口資料）。

### 11. 用戶
管理可登入後台的**管理員帳戶**（新增 / 編輯 / 啟用停用）。

### 12. 導師
導師完整檔案（編號、地區、任教科目、電話、SCRC、DSE、學歷、經驗、評語、電郵、性別、出生日期、身份證、地址、收款人、銀行）。可新增 / 編輯，可用最多 2 個條件搜尋，分頁顯示。

## 三、功能介紹（導師打卡系統 — 手機）

- **打卡**：只顯示**當天**要上的課堂；按「到達打卡」/「離開打卡」。系統記錄時間 + GPS，並按薪金規則計算堂費。若學校已設打卡範圍，須在範圍內方可打卡。
- **工作確認書**：顯示派給該導師的確認書；可按「簽署」進行電子簽名（手寫 + 同意聲明）、查看詳情、下載 PDF。
- **我的紀錄**：累計堂費及每次打卡紀錄。

## 四、使用手冊（常用操作步驟）

### A. 管理員：由零開始建立課程
1. 登入 `/login`。
2. 左上「**+ 新增學年**」→ 輸入開始年份（如 2026 → 2026-2027）。
3. 右上「**+ 新增學校**」→ 填學校資料（如需打卡，填緯度 / 經度 / 最大可打卡距離）。
4. 學校下方「**+ 新增課程**」→ 填課程資料 → 保存。
5. 進入課程 →「課程基本信息」→「小組」→「**+ 新增小組**」（設星期、時間、年級等）。
6. 「**課堂詳情**」分頁 →「批量產生課堂」依小組時間表自動建立課堂。

### B. 派導師 / 製作工作確認書
1. 課程頁 →「**批量更新**」。
2. 選操作「更換導師」→ 選小組 → 勾選日期 → 選導師。
3. 按「**保存**」直接派導師；或填 JC 導師費後按「**製作 JC**」產生工作確認書（導師可上網簽署）。

### C. 更新教材 / 物資進度
- 方法一：「**工作計劃表**」→ 該課程「更新狀態」→ 選日期 + 狀態 → 保存。
- 方法二：「課堂詳情」逐堂編輯。
> 兩者會同步顯示於「課程進度表」。

### D. 出糧
1. 導師完成打卡（堂費已於打卡時鎖定）。
2. 「**糧單**」→ 選月份 → 查看每位導師小計及總計 → 下載 CSV / XLSX。

### E. 導師：打卡
1. 手機開 `/tutor-login` → 輸入電話 → 「發送驗證碼」→ 輸入收到的驗證碼 → 登入。
2. 「打卡」分頁 → 當天課堂按「**到達打卡**」；下課按「**離開打卡**」。

### F. 導師：簽署工作確認書
1. 「工作確認書」分頁 → 找未簽署的一張 → 按「**簽署**」。
2. 在方框手寫簽名 → 勾選「本人同意接任以上工作安排」→ 「提交」。
3. 狀態轉為「已簽署」，可「下載 PDF」。

## 五、薪金規則（出糧）

- 每堂固定薪金，取自導師的「每堂薪金」。
- **提早 10 分鐘或以上打卡** → 100% 薪金。
- **遲於此時間** → 扣 70%，導師得 30%。
- 代課導師以**其本人**的每堂薪金計算。
- 堂費於打卡一刻**鎖定快照**，日後改 rate 不影響舊紀錄。

## 六、課程卡狀態圓點

| 圓點 | 意思 |
|---|---|
| 🔴 紅 | 任何一堂**未有課程內容** |
| 🟡 黃 | 任何一堂**未有導師**（空缺） |
| 🟣 紫 | 任何一堂**未有教材物資** |
| 🔵 藍 | **invoice 未完成** |

全部處理好則不顯示圓點。

## 七、注意事項（正式上線前）

1. **SMS 驗證碼**：目前為開發模式，驗證碼會顯示在畫面（方便示範）。正式給導師使用前**必須**接駁真實 SMS 供應商，否則任何人知道電話號碼即可登入。
2. **身份證號碼**：目前明文儲存，屬敏感個人資料（PDPO），正式輸入真實資料前須加密。
3. **方案**：示範用 Vercel 免費版（非商業用途）。正式向客戶收費時應轉用付費方案，並將 Supabase 升級至 Pro（每日備份）。
